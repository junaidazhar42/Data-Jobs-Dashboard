# Data Jobs Dashboard with Power BI

![Dashboard Page 1](/images/page1.png)

## Introduction
This dashboard is for **Job Seekers, Job Transitioners and Job Swappers**. Information about the data job market is scattered and the dashboard tries to solve this problem.
This project uses real-world dataset consisting of 2024 data science job postings and provides an easy-to-use interface to explore the data job market.


## Skills Showcased
- **Data Transformation (ETL) with Power Query**: Cleaned and prepared the raw data through changing data types, adding new columns and dealing with blank values.
- **Implicit Measures**: Created measures for KPIS like Median Yearly Salary, Median Hourly Salary and Job Count. 
- **Charts and Geospatial Analysis**: Created Column, Bar, Line and Area Charts to track job market trends. Made use of Map Charts to get global distribution of jobs.
- **KPI Indicators & Tables**: Cards to display important metrics and Tables for granular data. 
- **Dashboard Design**: Designed an appealing layout, making use of common and uncommon plots to communicate the data job market effectively.
- **Interactive Reporting**:
  - **Slicers**: Dynamic filtering by Job Title.
  - **Buttons & Bookmarks**: To create a seamless navigation experience.
  - **Drill-Through**: To navigate from a high-level view to a detailed summary. 

## Dashboard Overview
The dashboard is split into two distinct pages.

### Page 1: High-Level Market View
![Dashboard Page 1](/images/page1.png)
This is the first page of the dashboard. It displays key KPIs like total job count, average job rating, median yearly and hourly salaries and lets job seekers get a quick
glance of what's going on in the job market.

### Page 2: Job Title Drill Through
![Dashboard Page 1](/images/page2.png)
Users can drill through to this page from the main dashboard to find out details for a single job title like 'Data Analyst', 'Data Engineer' etc. This page includes information 
like yearly and hourly salaries, job details like work from home and health insurance perks, platforms on which the jobs were listed as well as a global map of job locations.
